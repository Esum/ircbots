# ircbots

lyrbot
------

Un bot qui vous fait deviner des titres de chansons.

Les paroles sont dans `paroles/`.

Format :

```
== Chanteur

= Titre = id_youtube = nombre_de_vers_à_afficher = mot_blacklisté | mot_blacklisté = notes

Bla bla bla
Bla bla bla

Deuxième strophe
Bla bla bla

= Titre = ...

== Chanteur ...

...

=
```

(n'oubliez de terminer le fichier par un `=`)
